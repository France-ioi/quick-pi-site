Configuration Nous vous proposons un ensemble de tutoriaux pour configurer votre matériel selon votre situation.
1. Préparation d'un Raspberry Pi

Attention : passez directement à l'étape 3 si vous avez acheté notre kit, il est livré déjà monté et installé.

Si au contraire vous achetez vos composants séparément, et en particulier votre propre Raspberry Pi, voici quelques explications sur comment les préparer :

Tutoriel sur la préparation du Raspberry Pi Zero.

Tutoriel sur la préparation du Raspberry Pi 3A+.

Si vous achetez vos LEDs Grove, suivez nos informations sur la préparation des LEDs, pour gagner un peu de temps.

2. Installation de la carte micro-SD

Attention : ignorez cette étape si vous avez acheté notre kit, il est déjà installé.

Sinon, vous devez installer le système d'exploitation et les outils sur la carte micro-SD (minimum 8GO) de votre Raspberry Pi.

Suivez pour cela notre tutoriel d'installation de la carte micro-SD.

3. Carte QuickPi : navigation dans le menu et test

Si vous disposez de la carte QuickPi ou de notre Kit complet, vous pouvez commencer par découvrir comment naviguer dans son menu et lancer l'autotest, qui vérifie que l'ensemble des capteurs et actionneurs fonctionnent.

Si vous n'avez pas de carte QuickPi, passez à l'étape suivante.

4. Configuration de la connexion

Une fois votre matériel prêt, l'étape suivante est de configurer la ou les manières dont vous allez connecter votre Raspberry Pi à votre ordinateur, votre réseau Wifi, et si possible à Internet.

Suivez pour cela notre tutoriel de configuration.

5. Environnement de programmation

Vous pouvez ensuite charger l'environnement de programmation à votre Raspberry Pi.

Depuis cet environnement, vous pourrez vous connecter au Raspberry Pi pour y exécuter vos programmes. Suivez pour cela le tutoriel de connexion.
