Configuration Nous vous proposons un ensemble de tutoriaux pour configurer votre matériel selon votre situation.
1. Installation de la carte micro-SD

Sauf si vous achetez notre kit livré pré-installé, vous devrez installer le système d'exploitation et les outils sur la carte micro-SD (minimum 8GO) de votre Raspberry Pi.

Suivez pour cela notre tutoriel d'installation de la carte micro-SD.

2. Connexion Wifi

Pour connecter votre matériel à l'application QuickPi, nous vous conseillons d'utiliser le Wifi.

Suivez pour cela notre tutoriel de configuration Wifi.

3. Connexion USB

Si vous n'avez pas la possibilité d'utiliser un réseau Wifi, il est possible de configurer certains Raspberry Pi, en particulier le Raspbery Pi Zero WH, pour se connecter à internet au travers d'une connexion USB. Ceci fonctionne sur les systèmes d'exploitation récents, notamment à partir de Windows 7, mais peut nécessiter une installation de drivers.

Suivez pour cela notre tutoriel sur la connexion par USB.

4. Connexion bluetooth

Depuis une tablette, et en particulier un iPad, il est possible de se connecter à votre Raspberry Pi en Bluetooth.

Suivez pour cela notre tutoriel sur la connexion par Bluetooth.

5. Préparation d'un Raspberry Pi

Si vous achetez vos composants séparément, et en particulier votre propre Raspberry Pi, voici quelques explications sur comment les préparer :

Tutoriel sur la préparation du Raspberry Pi Zero.

Tutoriel sur la préparation du Raspberry Pi 3A+.

Notez que si vous choisissez notre kit, il est livré déjà monté et installé.

6. Préparation des LEDs Grove

Si vous achetez vos LEDs Grove, suivez nos informations sur la préparation des LEDs, pour gagner un peu de temps.
