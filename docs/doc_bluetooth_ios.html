Connezion Bluetooth depuis un iPad ou iPhone

Pour pouvoir vous connecter par Bluetooth à votre Raspberry Pi, commencez par activer cette possibilité depuis la page de configuration de celui-ci (voir plus haut). Une fois que c'est fait, le fonctionnement dépend de votre appareil.

Allez dans la page de configuration de votre iPad ou iPhone, et activez le Bluetooth. Vous voyez alors la liste des appareils bluetooth disponibles, qui ressemble à ceci :

Dans la zone "connexion aux autres appareils", vous devriez y voir apparaître "quickpi1", ou le nom que vous avez donné à votre QuickPi. Cliquez dessus. Au bout d'un moment, il devrait apparaître comme connecté. Réessayez si cela ne fonctionne pas dès la première fois.

Vous pouvez alors vous connecter à ce Raspberry depuis l'environnement QuickPi en utilisant l'onglet "Bluetooth" dans son interface de connexion.

Vous pouvez aussi charger la page de configuration directement dans votre navigateur, en allant à l'adresse : 192.168.233.2

Depuis un téléphone ou une tablette Android

Il est possible de se connecter par Bluetooth au Raspberry Pi depuis un appareil Android, mais cela nécessite de désactiver le Wifi, donc votre connextion internet.

Depuis la page de configuration de votre appareil Android, allez sur la page Connexions. Désactivez le wifi, puis activez le Bluetooth.

Dans la zone "Appareils disponibles", vous devriez y voir apparaître "quickpi1", ou le nom que vous avez donné à votre QuickPi. Cliquez dessus. Au bout d'un moment, il devrait apparaître comme connecté. Réessayez si cela ne fonctionne pas dès la première fois.

Une fois connecté, allez ensuite dans la configuration de l'appareil, puis activez son utilisation pour l'accès internet.

Vous n'aurez plus accès à internet, mais vous pourrez, depuis votre navigateur, accéder à la page web de votre Raspberry Pi. Pour cela, entrez 192.168.233.2 dans votre navigateur.
