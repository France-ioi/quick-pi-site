Connecter son Raspberry par Wifi

La connexion par Bluetooth est utile lorsque la connexion USB n'est pas disponible, par exemple sur les tablettes.

Ouverture du fichier de configuration

Pour ouvrir le fihier situé sur la carte micro-SD, suivez la procédure "Ouvrez quickpi.txt".

Activer le bluetooth

Modifiez la ligne ENABLEBLUETOOTH=0 en remplaçant le 0 par un 1, pour activer le Bluetooth.

Enregistrez le fichier, et éjectez proprement la carte Micro-SD avant de la replacer dans le Raspberry Pi éteint.
