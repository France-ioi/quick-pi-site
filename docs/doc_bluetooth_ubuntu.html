Connexion Bluetooth depuis Linux - Ubuntu

Pour pouvoir vous connecter par Bluetooth à votre Raspberry Pi, commencez par activer cette possibilité depuis la page de configuration de celui-ci (voir plus haut). Une fois que c'est fait, le fonctionnement dépend de votre appareil.

Cliquer sur le menu en haut à droite, puis sur l'icône bluetooth

La liste des appareils s'affiche, quickpi1, ou le nom que vous avez donné à votre Raspberry Pi, devrait apparaître après quelques secondes.

Cliquez dessus, la connexion s'effectue, puis après quelques secondes, cela affiche "Connected" puis "Disconnected".

Fermez la fenêtre Bluetooth (cela ne fonctionne pas si vous ne la fermez pas), puis dans le menu en haut à droite, sélectionner "Connect to Internet" en-dessous de la connexion à quickpi1 (ou le nom que vous avez choisi).

Après quelques secondes un triangle vert apparaît en haut à droite pour indiquer que vous êtes bien connecté.

Vous pouvez alors vous connecter à ce Raspberry Pi depuis l'environnement QuickPi en utilisant l'onglet "Bluetooth" dans son interface de connexion.

Vous pouvez aussi charger la page de configuration directement dans votre navigateur, en allant à l'adresse : 192.168.233.2
